- **超卖区**：RSI < 30（极度超卖 < 20）

- **跌破下轨**：%B < 0
- **下轨至中轨**：0 ≤ %B < 0.5
- **中轨至上轨**：0.5 ≤ %B ≤ 1.0
- **突破上轨**：%B > 1.0

#### MACD

- **死叉**：MACD柱 < 0且MACD线 < 信号线
